Een intake systeem waar templates gebruikt kunnen worden en waar na het maken van de intake het systeem automatisch herinneringen stuurt.

Mijn rol was alle backend functionaliteit van het project faciliteren. Een simpele monoliet CRUD-systeem was genoeg gezien het alleen gebruikt zou worden door 2 tot 5 mensen tegelijkertijd. De notificaties worden gestuurd vanuit de server naar whatsapp. Het bedrijf gebruikt 1 telefoon nummer voor alle intake zaken dus de server hoefde alleen toegang te hebben tot 1 whatsapp.
